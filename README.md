1. What is the difference between var, let, and const?
Ans: var is the old way to declare variables in JavaScript. It has function scope and can be redeclared, which can cause bugs. That's why we now use let and const. let is used when the value needs to change, and const is used when the value should stay the same.

2. What is the spread operator (...)?
Ans:Spread operator is used for arrays and objects. It spreads the values of an array or object. If I use spread operator to copy an array, the original array won't be changed.

3. What is the difference between map(), filter(), and forEach()?
Ans: map() is used when i want to return a new array with changed values. filter() is used when i want to return a new array with only some values that match a condition. forEach() is just for looping, it don't return anything.

4. What is an arrow function?
Ans: Arrow function is a short way to write a function in JavaScript. Instead of writing the function keyword we use =>.

5. What are template literals?
Ans: Template literals is a way to write string with variables inside it. Instead of using + to join strings we use backtick and ${} to put variables inside the string.
